  ## Deploying to GitHub Pages

  This repository is set up with a GitHub Actions workflow in `.github/workflows/deploy.yml`.

  If your GitHub handle is `raziasultanajui2`, your GitHub Pages user site address will be `raziasultanajui2.github.io`.

  You can still use `JUI` as the site title, brand name, or repository name.

  To publish it as a site like `raziasultanajui2.github.io`:

  1. Create a GitHub repository named `raziasultanajui2.github.io`.
  2. Push this project to the `main` branch.
  3. In GitHub, open `Settings` -> `Pages` and set the source to `GitHub Actions`.
  4. Push any change to `main`; the workflow will build and deploy automatically.

  If you use a custom domain, add it in GitHub Pages settings and keep the Vite base path as `/`.
